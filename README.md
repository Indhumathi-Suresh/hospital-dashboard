**Hospital Dashboard**
**Project Description**
This project analyzes hospital operational, clinical, and financial performance using Tableau dashboards. It helps hospital management monitor admissions, efficiency, costs, and revenue across departments and branches.
**Dashboards**
1.	Hospital Executive Overview – High-level KPIs (Admissions, ALOS, Bed Occupancy, Cost, Readmissions)
2.	Operational Efficiency & Department Performance – Department-wise admissions, ALOS, bed occupancy, emergency analysis
3.	Financial Performance & Cost Analysis – Revenue, costs, insurance coverage, patient payments
Tools Used****
•	Tableau Desktop & Tableau Public
•	CSV datasets
•	GitHub
